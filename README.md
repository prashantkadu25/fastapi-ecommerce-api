🚀 FastAPI E-commerce API
📌 Project Overview
This is a production-ready FastAPI backend project with:

User authentication (JWT)
CRUD operations
Pydantic validation
SQLAlchemy database integration
Deployed on AWS EC2


⚙️ Tech Stack

FastAPI
Python
SQLAlchemy
Pydantic
JWT Authentication
AWS EC2


🚀 Features

User Registration
User Login (JWT Token)
Create / Read / Delete Users
Input validation using Pydantic
REST API structure


🌐 Live API
http://43.205.192.148:8000/docs


▶️ Run Locally
git clone https://github.com/prashantkadu25/fastapi-ecommerce-api.git
cd fastapi-ecommerce-api

python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt
uvicorn main:app --reload
